## STARTING OFF:

Make sure you have npm installed.
Run:
```
cd app
npm install
npm install http-server -g
webpack
http-server
```

View at: http://localhost:3000/

## CONCEPTS:

TypeScript - a typed superset of JavaScript that compiles to plain JavaScript
